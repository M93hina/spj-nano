# Airoco × Jetson Nano 環境モニタリングシステム 概要

## 1. 現在の機材・センサー状況

### Airoco(固定設置・学内実験棟)
温度・湿度・CO2濃度を計測できるIoTセンサー。以下の箇所に設置済み。

| 設置場所 | 種別                   | 備考                                                           |
| -------- | ---------------------- | -------------------------------------------------------------- |
| B1F      | EH(エレベーターホール) | フリースペース全体を含む。机・椅子・カウンター・コンビニ等あり |
| 1F       | EH                     | -                                                              |
| 3F       | EH                     | -                                                              |
| 4F       | EH                     | -                                                              |
| 301      | 3F 講義室              | 比較的大きめの部屋                                             |
| 401      | 4F 講義室              | -                                                              |
| 403      | 4F 講義室              | -                                                              |

- 301, 401, 403は講義・実験用の部屋だが、使用頻度はそこまで高くない
- 各階には上記以外にも研究室等が存在

### Jetson Nano 関連機材
- **Jetson Nano本体**:実験室内での使用に限定(持ち出し不可)
- **BME280センサー**:温湿度・気圧計測。Jetson Nanoに接続して使用
- **Webカメラ**:画像認識(人数カウント等)に使用可能

### 制約条件
- 学内施設のため、カメラの長時間設置は不可
- Jetson Nano本体は実験室外への持ち出し不可

上記制約により、カメラ・Jetson Nanoは実験室内に閉じた検証・開発用途に限定し、学内全体をカバーする本番システムはAirocoのデータのみで構成する方針とする。

---

## 2. システム構想の概要

### 基本方針
- **本番システム**:Airocoの既存センサーデータのみで学内全体をカバー(新規設置作業不要)
- **裏方(実験室内)**:Jetson Nano + BME280 + Webカメラを用いて、CO2データから人数を推定するモデルを学習・検証し、その知見(モデル)のみを本番システムに輸出する

### 有力な方向性:B1Fの混雑予測システム

B1Fは学生が自由に使えるフリースペースであり、混雑状況の可視化ニーズが最も現実的。

**基本アイデア**
CO2濃度の変化から人の存在・増減を推定し、B1Fの混雑度をリアルタイムで可視化する。

**発展形:先行指標を用いた予測型混雑システム**
301・401・403のCO2下降(講義終了のサイン)や、3F・4FのEHのCO2スパイク(人の移動)は、B1Fが混み始める数分前の予兆として観測できる。これらの時系列データを組み合わせることで、「現在の混雑状況」ではなく「数分後の混雑予測」を提示するシステムへ発展させられる。


### Jetson Nano側の役割(実験室内)
- Webカメラによる人数カウント(YOLO等の物体検出)と、BME280・CO2データを同時記録
- 環境データと実際の人数の対応関係を学習し、CO2ベースの人数推定モデルを構築
- 学習済みモデルのロジックのみを本番システムへ組み込む
- 副次的に、実験室自体の環境モニタリング・安全用途にも活用可能

## 3. まとめ

| 要素                                    | 役割                                                                     |
| --------------------------------------- | ------------------------------------------------------------------------ |
| Airoco(全設置箇所)                      | 本番システムのデータソース。学内全体の環境・混雑モニタリング             |
| Jetson Nano + カメラ + BME280(実験室内) | CO2-人数推定モデルの学習・検証環境                                       |
| B1F混雑予測                             | 最有力候補。上階センサーを先行指標として活用した予測型システムへ発展可能 |
