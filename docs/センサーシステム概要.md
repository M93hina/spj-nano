# Airoco × Jetson Nano 環境モニタリングシステム 概要

## 1. 現在の機材・センサー状況

### Airoco(固定設置・学内実験棟)
温度・湿度・CO2濃度を計測できるIoTセンサー。以下の箇所に設置済み。

| 設置場所 | 種別                   | 備考                                                           |
| -------- | ---------------------- | -------------------------------------------------------------- |
| B1F      | EH(エレベーターホール) | フリースペース全体を含む。机・椅子・カウンター・コンビニ等あり |
| 1F       | EH                     | -                                                              |
| 3F       | EH                     | -                                                              |
| 4F       | EH                     | -                                                              |
| 301      | 3F 講義室              | 比較的大きめの部屋                                             |
| 401      | 4F 講義室              | -                                                              |
| 403      | 4F 講義室              | -                                                              |

- 301, 401, 403は講義・実験用の部屋だが、使用頻度はそこまで高くない
- 各階には上記以外にも研究室等が存在

### Jetson Nano 関連機材
- **Jetson Nano本体**:実験室内での使用に限定(持ち出し不可)
- **BME280センサー**:温湿度・気圧計測。Jetson Nanoに接続して使用
- **Webカメラ**:画像認識に使用可能

### 制約条件
- 学内施設のため、カメラの持ち出し・長時間設置は不可
- Jetson Nano本体は実験室外への持ち出し不可

上記制約により、カメラ・Jetson Nanoは実験室内に閉じた検証・開発用途に限定し、学内全体をカバーする本番システムはAirocoのデータのみで構成する方針とする。
